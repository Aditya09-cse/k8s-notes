# 🚀 K8s Notes - Simplified Kubernetes Learning Repository

Welcome to **K8s Notes** — a beginner-friendly GitHub repository designed to help students, DevOps learners, and cloud enthusiasts understand Kubernetes concepts in a simple and structured way.

This repo contains well-organized **Markdown notes** covering Kubernetes fundamentals along with practical topics like deployments, services, probes, namespaces, and more.

---

## 📌 Why This Repository?

Kubernetes can feel complex when starting out. This repo is made to:
- simplify Kubernetes concepts
- provide structured notes for revision
- help learners understand real-world Kubernetes components
- support hands-on learning

---

## 📂 Repository Structure

Here are the main files and what they cover:

### 📘 Kubernetes Basics
- **`k8s-intro.md`**  
  Introduction to Kubernetes and why it is used.

- **`core-concepts.md`**  
  Covers core Kubernetes concepts like Pods, Nodes, Cluster, and architecture basics.

---

### ⚙️ Kubernetes Architecture
- **`control-plane.md`**  
  Explains Kubernetes Control Plane components like API Server, Scheduler, Controller Manager, etc.

---

### 🚀 Workloads & Scaling
- **`deployments-replicasets.md`**  
  Explains Deployments, ReplicaSets, scaling, rolling updates, and rollback concepts.

- **`pod-lifecycle.md`**  
  Covers pod lifecycle stages, restart policies, and pod behavior.

---

### 🔑 Configuration & Storage
- **`configmaps-secrets-volumes.md`**  
  Explains how to manage environment configs, secrets, and volumes in Kubernetes.

---

### 🏷️ Organization & Management
- **`labels-selectors-namespaces.md`**  
  Explains labels, selectors, namespaces, and how Kubernetes resources are organized.

---

### 🌐 Networking & Access
- **`services.md`**  
  Covers Kubernetes services (ClusterIP, NodePort, LoadBalancer) and how apps are exposed.

---

### ❤️ Health Monitoring
- **`probes.md`**  
  Explains Liveness Probe, Readiness Probe, and Startup Probe.

---

### ⚡ Quick Reference & Practice
- **`cheatsheet.md`**  
  Contains useful Kubernetes commands for quick revision.

- **`hands-on-project.md`**  
  Practical project ideas and exercises to build real Kubernetes understanding.

---

## 🎯 Who Is This Repo For?

This repository is helpful for:
- Beginners starting Kubernetes
- DevOps students preparing for interviews
- Cloud learners working on container orchestration
- Anyone who wants quick revision notes

---

## ⭐ How to Use This Repo

1. Start with `k8s-intro.md`
